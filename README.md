# Backrooms Online

Welcome to **Backrooms Online**, a survival horror game inspired by the eerie and unsettling atmosphere of the Backrooms. Navigate through an endless maze of monotonous yellow walls, flickering fluorescent lights, and an overwhelming sense of dread. Can you escape, or will you be lost in the Backrooms forever?

## Features

- **Procedurally Generated Levels**: No two runs are ever the same, ensuring a fresh experience every time you play.
- **Dynamic AI**: Encounter unpredictable entities that adapt to your play style, keeping you on edge.
- **Immersive Atmosphere**: Ambient sounds, lighting, and environmental storytelling bring the Backrooms to life.
- **Resource Management**: Scavenge for supplies and tools while avoiding detection.

## Gameplay

- **Explore**: Navigate through procedurally generated rooms and hallways.
- **Survive**: Avoid entities that lurk within the Backrooms. Stealth and quick thinking are your best allies.

## Controls

- **W/A/S/D**: Move
- **Mouse**: Look around
- **Shift**: Sprint
- **Ctrl**: Crouch
- **E**: Interact
- **F**: Use flashlight (if found)
- **Tab**: Open inventory

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/backrooms-explorer.git
   ```
2. Navigate to the project directory:
   ```bash
   cd backrooms-explorer
   ```
3. Import into Godot editor
